Get a user's profile

This endpoint allows you to retrieve information about a user's profile.

Endpoint

GET https://debuggerduck.api.com/users/{user_id}

Request parameters

Header fields

Header field

Value description

Authorization

A valid access token. Required.

Path parameters

Path parameter

Value description

user_id

The user's ID. Required.

Sample request

curl -X GET "https://debuggerduck.api.com/users/{user_id}" 
-H "Authorization: Bearer {your access token}"

Response parameters

See the Response codes and the sample User object below.

User object

Key

Value type

Value description

name

string

The user's name displayed on their profile.

id

string

The user ID for this user.

href

string

The Web API endpoint link for this user.

img

image object

The user's profile image.

Sample User object


{
"name" : "Anthon Anthony",
"href" : "https://debuggerduck.api.com/users/anthon11",
"id" : "anthon11",
"images" : [ {
   "height" : null,
   "url" : "http://debuggerduck.com/profileimages/anthon11/img.png",
   "width" : null
} ]
}

Send a message

This endpoint allows you to send messages to a DebuggerDuck.

Endpoint

POST https://debuggerduck.api.com/messages

Request parameters

The body of the HTTP request is a JSON. All values are required.

Key

Value type

Value description

sender_id

string

The user ID for this user. Required.

message

string

Contains the user's message. Required.

Sample request


curl -X POST -H "Content-Type: application/json" -d '{
"sender_id": "id",
"message":"hello, duck!"
}' "https://debuggerduck.api.com/messages?access_token={your access token}"
