Getting started

Create simple apps to communicate with your DebuggerDuck. Here's what you need to get started.

Register your app with FullStacks, Inc.

  • To register your app, fill out the form at debuggerduck.com/developers

Prepare your environment

  1. Set up the server and software platform of your choice.
  2. Test your development environment.

Get an access token

  1. From our website, download Duckman and install it.
  2. In Duckman, make the following request:
    POST https://debuggerduck.api.com/oauth2/token
  3. On the Authorization tab, select Basic.
    1. In User ID, enter your user ID.
    2. In Password, enter your secret.
  4. Select Send.
    You receive your access token from the Duckman server:
    {
"scope": "scope",
"access_token": "your-token",
"token_type": "bearer",
"app_id": "ABC-123456",
"expiry": 12345,
}
  5. In the Authorization header of your requests, include the following:
    "Authorization: bearer your-token"

Make requests to the API endpoints
