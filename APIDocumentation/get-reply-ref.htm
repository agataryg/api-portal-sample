Get a reply from a DebuggerDuck

This endpoint allows you to retrieve replies from a DebuggerDuck by their IDs.

Endpoint

GET https://debuggerduck.api.com/messages/{user_id}/received/{id}

Request parameters

Header fields

Header field

Value description

Authorization

A valid access token. Required.

Path parameters

Path parameter

Value description

user_id

The user's user ID. Required.

id

The ID of the message. Required.

Sample request

curl -X GET "https://debuggerduck.api.com/messages/{user_id}/received{id}" 
-H "Authorization: Bearer {your access token}"

Response parameters

See the Response codes and the sample Reply object below.

Response object

Key

Value type

Value description

date

long

The message creation timestamp, used to order the incoming replies.

blinked

boolean

Information about the DebuggerDuck's reaction to a message.

Sample Response object


{
"date" : "1472799302",
"blinked" : true
}
